# moopy 
Simple application using Django and Semantic-UI

version : 1.0

- create database name 'moopy'
- virtualenv .
- pip install requirements.txt
- settings.py

```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'moopy',
        'USER': 'root',
        'PASSWORD': '12345678',
    }
}
```

url home : http://localhost:8000/movies/
